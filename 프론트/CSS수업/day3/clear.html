- clear는 float 속성과 같이 사용 - none : float 속성이 적용된 요소 위치에 상관없이 내용들이 중간에 들어가도록함 - left : float 속성이 left로 적용된 요소 위치 다음부터 내용들이 들어가도록 함 - right : float 속성이 right로 적용된 요소 위치 다음부터 내용들이 들어가도록 함 - both : float 속성이 left와 right로 적용된 요소 위치 중 가장 밑에 있는 요소를 기준으로 다음부터 내용들이 들어가도록 함 .text 부분에 clear값을 바꿔가면서 확인하는것이 빠름
